# 复古风格博客网站 - Web课程作业
这是一份Web课程作业成果，实现了一个**复古风格个人学习博客网站**，整体设计充满怀旧感，功能聚焦于技术学习记录与小组信息展示，满足课程作业的展示与实用需求。

## 一、 网站核心功能板块
1.  **学习笔记分享板块**
    -  核心用于记录前端、Linux、Web（JavaScript）等技术方向的学习实践心得
    -  支持呈现“网页添加独立评论功能”“Nginx部署静态网页”等实操类技术笔记
    -  笔记内容以简洁短讯形式排版，搭配复古风格边框样式，视觉上贴合整体主题
2.  **小组信息展示板块**
    -  包含“关于我们”专属模块，清晰罗列小组成员信息及对应头像
    -  网站底部配置完整联系方式，涵盖小组邮箱、GitHub项目链接等，方便作业展示与交流

## 二、 复古风格设计特点
-  布局：采用经典复古网页的简洁架构，搭配导航栏+区块式内容排版，还原早期网页的视觉体验
-  样式：使用怀旧感字体、复古风边框装饰，色彩搭配贴合复古主题，弱化繁杂特效，突出实用性
-  定位：功能聚焦学习记录与小组作业展示，无冗余复杂功能，完全契合Web课程作业的核心要求

## 三、 作业相关说明
-  项目类型：Web课程实践作业
-  小组组成：梁锦涛等7位成员
-  核心用途：技术学习笔记记录、小组作业成果展示
