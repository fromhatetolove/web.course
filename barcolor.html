基础JS应用：导航栏闪烁

发布时间：2025-12-24 | 作者：梁锦涛

一、功能说明

实现导航栏每0.05秒自动切换背景颜色，同时同步更新.about-section h2与.contact-section h2的下边框颜色，形成统一视觉效果。

document.addEventListener('DOMContentLoaded', function() {
    const navBar = document.querySelector('.nav-bar');
    const targetH2List = document.querySelectorAll('.about-section h2, .contact-section h2');
    if (!navBar) return;
    const colorList = ['#2c3e50', '#34495e', '#1abc9c', '#2ecc71', '#3498db', '#9b59b6', '#e67e22', '#e74c3c', '#16a085', '#27ae60'];
    let colorIndex = 0;
    setInterval(function() {
        colorIndex = (colorIndex + 1) % colorList.length;
        const currentColor = colorList[colorIndex];
        navBar.style.backgroundColor = currentColor;
        targetH2List.forEach(h2 => {
            h2.style.borderBottomColor = currentColor;
        });
    }, 50);
});

三、关键要点

1. 使用document.querySelector获取导航栏，document.querySelectorAll批量获取目标h2元素

2. colorList预设多种颜色值，支持十六进制格式

3. (colorIndex + 1) % colorList.length实现颜色循环切换，避免下标越界

4. setInterval设置50毫秒定时器，实现定时切换

5. borderBottomColor同步更新h2下边框颜色，与导航栏样式统一

四、效果验证

页面加载后，导航栏将自动开始颜色切换，h2下边框颜色同步变化，无需额外操作，兼容无.about-section或.contact-section的页面场景。
