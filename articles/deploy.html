【Linux学习】Nginx部署静态网页

发布时间：2025-12-16 | 作者：梁锦涛

一、准备工作

1. 拥有一台Linux电脑

2. 安装Nginx：sudo (包管理器) install nginx -y

3. 启动Nginx：sudo systemctl start nginx

二、部署步骤

1. 将博客文件放到 /usr/share/nginx/html 目录下

2. 重启Nginx：sudo systemctl restart nginx

3. 访问服务器IP，即可看到博客页面

三、总结

Linux下部署静态博客超级简单，这就是我们的学习记录，希望对大家有帮助~
