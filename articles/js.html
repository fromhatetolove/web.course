一、什么是JS

JS（JavaScript）是一门运行在浏览器中的脚本语言，主要用于给网页添加交互效果，比如按钮点击、内容切换、数据验证等，是Web前端三大核心技术（HTML/CSS/JS）之一。

二、JS基础语法（简单示例）

JS代码可以直接写在HTML的<script>标签中，也可以外部引入（我们这里用内嵌方式，更易入门）。

示例3：按钮点击切换背景颜色

四、示例对应的JS代码解释

上面的按钮交互对应的JS代码如下，所有功能都简单易懂：

// 示例2：修改文字的函数
function changeText() {
    // 获取id为textBox的元素，修改它的文字内容
    document.getElementById("textBox").innerText = "我被JS修改啦！是不是很神奇~";
}

// 示例3：切换背景颜色的函数
function changeBgColor() {
    // 定义一组颜色值
    const colors = ["#f0f8ff", "#f5fafe", "#e8f4f8", "#faf0e6", "#f5f5dc"];
    // 随机获取一个颜色索引
    const randomIndex = Math.floor(Math.random() * colors.length);
    // 获取文章详情区域，修改背景色
    document.querySelector(".article-detail").style.backgroundColor = colors[randomIndex];
}

五、JS基础总结

1. JS的核心作用是实现网页交互，让静态页面“动”起来

2. 按钮点击是最常用的交互场景，通过`onclick`属性绑定JS函数即可实现

3. `document.getElementById()`和`document.querySelector()`是获取页面元素的常用方法

4. 通过`innerText`修改文字内容，通过`style`属性修改元素样式
